Sideloading Instructions

After downloading the manifest file from GitHub, follow the steps for your operating system to sideload Obiter into Word.

Mac (Word 2024 / Microsoft 365)

  1. Download the manifest.xml file from the latest GitHub release.
  2. Open Finder and press Cmd + Shift + G to open the Go to Folder dialog.
  3. Paste this path: ~/Library/Containers/com.microsoft.Word/Data/Documents/wef
  4. If the wef folder does not exist, create it.
  5. Copy manifest.xml into the wef folder.
  6. Open Word and go to Insert > My Add-ins > My Organization.
  7. Select Obiter and click Add.

Windows (Word 2024 / Microsoft 365)

  1. Download the manifest.xml file from the latest GitHub release.
  2. Open File Explorer and navigate to: \\%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\
  3. If the Wef folder does not exist, create it.
  4. Copy manifest.xml into the Wef folder.
  5. Open Word and go to Insert > My Add-ins > My Organization.
  6. Select Obiter and click Add.

Word for Web

  1. Download the manifest.xml file from the latest GitHub release.
  2. Open a document in Word for Web at office.com.
  3. Go to Insert > Office Add-ins.
  4. Select Upload My Add-in in the upper-right corner.
  5. Click Browse and select the manifest.xml file.
  6. Click Upload. Obiter will appear in the Home tab.

System Requirements

Microsoft Word

Word 2024 or Microsoft 365 (desktop, web, or iPad).

Operating System

Windows 10 or later, macOS 12 (Monterey) or later, or any modern browser for Word for Web.

Internet

Required only for initial installation. Obiter works fully offline after that.

Frequently Asked Questions

Is Obiter free?

Yes. Obiter is free and open-source software, licensed under the GNU General Public License v3.0. You can use, modify, and distribute it freely.

Does it work on Word for Web?

Yes. Obiter supports Word for Web through the Office Add-ins platform. You can sideload it or install it from AppSource.

Does it work on iPad?

Yes. Obiter works on iPad through Microsoft Word for iPad, which supports Office Add-ins. Install from AppSource for the easiest setup.

Do I need an internet connection?

No. After the initial installation, Obiter works fully offline. All citation formatting, validation, and bibliography generation happens locally within your document.

What data does Obiter collect?

None. Obiter does not collect, transmit, or store any personal data. All citation data remains in your document. See the contact page for any privacy questions.
